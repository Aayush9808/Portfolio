## ğŸ¨ Customization

### Change Colors
Edit CSS variables in `styles.css`:
```css
:root {
    --primary: #00f0ff;     /* Cyan */
    --secondary: #7b2cbf;   /* Purple */
    --accent: #ff006e;      /* Pink */
}
```

### Add Your Photo
1. Place your image in `assets/images/profile.jpg`
2. Update the profile section in `index.html`

### Add Projects
Add new project cards in the projects section of `index.html`
